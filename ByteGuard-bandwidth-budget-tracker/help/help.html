How ByteGuard works

ByteGuard estimates downloaded data from completed browser requests, keeps your totals locally on-device, and gives you quick controls for staying under budget.

Settings Analytics Privacy policy

Start

Getting started

  • Open the popup to see today's usage and budget progress.
  • Visit Settings to set daily and monthly limits.
  • Turn on low-data mode when you want ByteGuard to block images and media.
  • Use Analytics to review trends and top sites.

Tracking

What is measured

ByteGuard reads request metadata such as the response size header when a request finishes. It uses that to estimate the amount of data downloaded by each site and tab.

Cached responses may contribute little or no additional usage because the browser did not need to download them again.

Controls

Low-data mode and blocking

Low-data mode blocks images and media across sites to reduce usage quickly. Domain blocking is stricter: it blocks requests to a specific site until you remove it from the blocked list.

Data

Privacy and exports

All settings and usage totals stay in your browser storage. You can export JSON backups, CSV history, and a weekly summary from the Settings page whenever you want.

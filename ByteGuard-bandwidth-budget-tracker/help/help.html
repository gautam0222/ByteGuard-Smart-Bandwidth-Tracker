Getting Started

What does ByteGuard do?

ByteGuard monitors how much data each website uses as you browse. It helps you:

  • Track bandwidth usage per tab and per website in real time
  • Set daily and monthly data budgets with smart alerts
  • Block data-heavy content automatically with Low-Data Mode
  • View usage trends with interactive charts and insights
  • Export data as JSON, CSV, or HTML reports

Quick Setup (2 minutes)

Step 1: Click the ByteGuard icon in your toolbar

Step 2: Click Settings in the bottom navigation

Step 3: Set your daily budget (e.g., 500 MB)

Step 4: Set your monthly budget (e.g., 10 GB)

Step 5: Click "Save Settings" — you're done!

💡 Tip:

Not sure what budget to set? Start with 500 MB/day and 10 GB/month, then adjust after a few days.

Features Guide

Real-Time Tracking

Your bandwidth is tracked automatically. The popup shows today's usage, budget remaining, top sites, and a progress ring.

Low-Data Mode

Save 70-90% bandwidth by blocking images and videos across all sites. Toggle it from the popup or let it auto-activate at 90% budget.

💡 Pro Tip:

Enable "Auto Low-Data Mode" in Settings to automatically block media when you hit 90% of your budget!

Site Blocking

Each site in the popup has a block button. Click it to block that domain completely. Manage blocked sites in Settings.

Charts & Analytics

  • Daily Trend: Usage over 7/14/30 days
  • Top Sites: Doughnut chart of data consumption
  • Budget Progress: Daily vs monthly comparison
  • Hourly Usage: When you use most data

Export & Backup

  • JSON: Complete backup of all data
  • CSV: Spreadsheet-friendly format
  • Weekly Summary: Beautiful HTML report
  • Import: Restore from any JSON backup

Troubleshooting

Extension not tracking data

  • Go to your browser's extensions page and click the reload button
  • Check the service worker for errors by clicking "service worker" link
  • Visit some websites and check the popup again

Low-Data Mode won't stay on

  • Remove and re-add the extension
  • Check service worker console for errors
  • Ensure declarativeNetRequest permission is granted

Charts not loading

  • Charts need at least a day of data to show meaningful results
  • Try refreshing the charts page
  • Check browser console (F12) for errors
⚠️ Still having issues?

Open the service worker console from your browser's extensions page and look for error messages.

Frequently Asked Questions

Does ByteGuard slow down my browsing?
No. ByteGuard is highly optimized, uses less than 50MB memory, and doesn't affect page load times.
Is my data sent to any server?
Absolutely not. All data stays on your device. ByteGuard is 100% privacy-first — we don't collect or transmit anything.
How accurate is the tracking?
Very accurate. We track all HTTP requests using native browser APIs. Cached content shows as 0 bytes (correct, since it's not re-downloaded).
Will Low-Data Mode break websites?
Most sites work fine — text loads normally. Image-heavy sites may look different, but functionality remains. You can always toggle it off.
Can I sync across devices?
Not yet automatically. You can export JSON on one device and import on another. Cloud sync is planned for a future update.
How do I reset everything?
Go to Settings → Data Management → "Clear All Data". This resets all usage statistics and settings to defaults.
Does this work with VPNs?
Yes! ByteGuard tracks at the browser level, not network level, so VPNs don't affect accuracy.

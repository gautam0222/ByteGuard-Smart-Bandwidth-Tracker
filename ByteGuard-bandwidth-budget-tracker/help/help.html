üöÄ Getting Started

What does this extension do?

Bandwidth Budget Tracker monitors how much data each website uses as you browse. It helps you:

  • Track bandwidth usage per tab and per website
  • Set daily and monthly data budgets
  • Get alerts when approaching limits
  • Block data-heavy content automatically
  • View usage trends with beautiful charts

Quick Setup (2 minutes)

Step 1: Click the extension icon in your toolbar

Step 2: Click "‚öôÔ∏è Settings" button

Step 3: Set your daily budget (e.g., 500 MB)

Step 4: Set your monthly budget (e.g., 10 GB)

Step 5: Click "Save Settings"

That's it! The extension is now tracking your usage.

üí° Tip: Not sure what budget to set? Start with 500 MB/day and 10 GB/month, then adjust based on your actual usage after a few days.

üîß Troubleshooting

Extension not tracking data

Solution:

  • Reload the extension: Go to edge://extensions/ ‚Üí Click reload
  • Check service worker: Click "service worker" link, look for errors
  • Visit some websites, then check the popup again

Low-Data Mode toggle won't stay on

Solution:

  • Completely remove and re-add the extension
  • Check service worker console for errors
  • Make sure declarativeNetRequest permission is granted

Block buttons not visible

Solution:

  • Make sure you have browsed some websites first
  • Right-click popup ‚Üí Inspect ‚Üí Check console for errors
  • Reload the extension completely

Charts not loading

Solution:

  • Check if Chart.js CDN loaded (look at Network tab)
  • Need at least a few days of data for meaningful charts
  • Try refreshing the charts page

Export buttons don't work

Solution:

  • Check if "downloads" permission is granted
  • Make sure downloads aren't blocked in browser settings
  • Check browser console (F12) for errors
‚ö†Ô∏è Still having issues? Open the service worker console (edge://extensions/ ‚Üí click "service worker") and look for error messages. This will help identify the problem.

‚ùì Frequently Asked Questions

Q: Does this extension slow down my browsing?
A: No! The extension is highly optimized and runs in the background. It uses less than 50MB of memory and doesn't affect page load times.
Q: Is my browsing data sent to any server?
A: Absolutely not! All data stays on your device. We don't collect, store, or transmit any of your information. This is a privacy-first extension.
Q: How accurate is the bandwidth tracking?
A: Very accurate! We track all HTTP requests using Chrome's native APIs. The only limitation is cached content (which shows as 0 bytes, which is correct since it's not re-downloaded).
Q: Can I use this on mobile?
A: Currently, this works on desktop browsers (Chrome, Edge, Brave). Mobile browser extensions have limited functionality, but we're exploring mobile options for the future.
Q: Will Low-Data Mode break websites?
A: Most websites will work fine! Text content loads normally. Some image-heavy sites (Instagram, Pinterest) may look different, but functionality remains intact. You can always toggle it off if needed.
Q: Can I sync data across devices?
A: Not automatically yet, but you can export to JSON on one device and import on another. Cloud sync is planned for a future update!
Q: How do I reset everything?
A: Go to Settings ‚Üí Data Management ‚Üí Click "Clear All Data". This will reset all usage statistics and settings to defaults.
Q: Does this work with VPNs?
A: Yes! The extension tracks bandwidth regardless of VPN usage. It measures data at the browser level, not network level.

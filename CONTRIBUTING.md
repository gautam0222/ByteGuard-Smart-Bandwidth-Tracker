# Contributing to Bandwidth Budget Tracker

Thank you for your interest in contributing! 

## How to Contribute

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/bandwidth-budget-tracker.git`
3. **Create a branch**: `git checkout -b feature/your-feature-name`
4. **Make changes** and commit: `git commit -m "Add your feature"`
5. **Push**: `git push origin feature/your-feature-name`
6. **Submit a Pull Request**

## Code Standards

- Use ESLint and Prettier (run `npm run lint` and `npm run format`)
- Write clear commit messages
- Add comments for complex logic
- Test in multiple browsers

## Reporting Issues

- Use GitHub Issues
- Include browser version and steps to reproduce
- Screenshots are helpful!

## Questions?

Open a discussion or issue on GitHub.
